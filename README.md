# Modal-Para-treinar
Aplicação básica apenas para praticar a execução do modal no JavaScript. 
<p>O script contém><p>
<p>* A adição e remição de classes para abrir e fechar o modal<p>
<p>* A adição de evento para abrir e fechar o modal ao clicar<p>
<p>* A adição de evento 'keydown', para fechar o modal ao apertar uma tecla do teclado<p>
<p><p>
<p><p>
<p>Tecnologias utilizadas<p>
<p>*HTML<p>
<p>*CSS<p>
<p>*JavaScript<p>
